UPER to JSON

POST /uper2json

This endpoint allows sending binary data encoded in uPER (unaligned Packed Encoding Rules) format and receiving a structured JSON response.

Request Body

The request body should contain binary data encoded in uPER format.

Example Request

POST /uper2json HTTP/1.1
Host: www.example.com
Content-Type: application/octet-stream

[binary data]

Server Response

The server will respond with a JSON object containing the following fields:

{
    "responseCode": "200",
    "encoding": "json",
    "data": "{\"header\": {
        \"protocolVersion\": 2,
        \"messageID\": 2,
        \"stationID\": 0
        }}"
}

This is a typical example of the response structure you can expect when interacting with the API.
